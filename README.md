# Mulu ReStore

Mulu ReStore is a modern, full-stack eCommerce platform for curated vintage and restored furniture.  
Built with **Next.js** and **MongoDB**, it combines a fast storefront with admin tools, analytics, and AI-assisted product storytelling.

---

## Features

### Storefront
- Product browsing with categories and search
- Product detail pages with image galleries
- Cart and wishlist
- Secure checkout (Stripe)
- User accounts: profile, addresses, order history

### Admin
- Product CRUD + image management (Cloudinary)
- Order management with status updates
- User management with roles (RBAC)
- Sales & inventory analytics dashboard
- AI product description generator (admin-only)

---

## Tech Stack

- **Next.js** (App Router)
- **MongoDB + Mongoose**
- **NextAuth.js** (authentication, RBAC)
- **Stripe** (payments)
- **Cloudinary** (images)
- **Tailwind CSS** (UI)
